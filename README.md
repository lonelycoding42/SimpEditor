# SimpEditor
## 功能设计

![c12b0ef14015bf7355a547085bc25367](https://github.com/user-attachments/assets/3755a14b-6b88-47f1-aaa5-0dc25bde925d)

本程序为命令行工具，拥有极其简陋的功能与UI，以及惨不忍睹的兼容性

h  -帮助
v  -浏览文件内容
i  -在行后插入新一行
d  -删除指定行
f  -查找指定短语
rl -替换指定行
rs -替换指定短语
m  -显示统计数据(行数与字符数)
s  -保存文件
a  -文件另存为
c  -重新加载新文件
q  -退出文本编辑器
